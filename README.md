6. Метод наименьших квадратов. (OpenMP) 
Задан набор точек и значения функции в этих точках.
С помощью метода наименьших квадратов определить полиномиальную функцию, максимально приближенную к данной (порядок полинома <=3)

a. На небольшом количесиве точек (points4.txt) показано, что алоритм находит точные коэффициенты полинома при отсутствии ошибки в данных. На небольшом количестве точек ускорение при использовании параллельного программирования не заметно.
b. На большом объеме данных (n=200000) показано, что использование параллельного программирования ускоряет вычисление коэффициентов матрицы A в ~2 раза. Причем, при увеличении количества потоков более 3 прирост производительности не заметен.
